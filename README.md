# Projekt - analiza efektywności turbin wiatrowych

Projekt przedstawia dane dotyczące pomiaru wiatru w grudniu 2009r. i analizuje je pod względem rozkładu prędkości, kierunku i potencjalnej wyprodukowanej energii. Pozwala on wyciągnąć informacje, które mogłyby służyć jako wstępna analiza tego, czy opłaca się zainwestować w kolejne turbiny wiatrowe w tej okolicy i w jaką stronę je ustawić.
